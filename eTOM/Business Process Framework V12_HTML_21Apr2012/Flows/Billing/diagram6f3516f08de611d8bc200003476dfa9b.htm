Billing Flow - Level 3 Scenario 1: Authorization request /balance reservation (enough balance)

Diagram Description

The end user places a call using a prepaid account. Events being generated are authorization request for the call with balance reservation. There is enough balance for the duration implied by the reservation request.
The flow below (see Figure 1) is repeated for the duration of the call.

Pre Condition

- The end user uses a prepaid account
- The end user places a voice call
- There is a sufficient balance for the duration of the call implied by the reservation request
