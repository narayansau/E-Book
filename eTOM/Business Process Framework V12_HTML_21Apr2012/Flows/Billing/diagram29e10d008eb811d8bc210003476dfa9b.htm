Billing Flow - Level 3 Scenario 1a: Authorization Request/Balance reservation (not enough balance)

Diagram Description

The end user places a call using a prepaid account, or the end user has placed a call already. Events being generated are authorization request for the call with balance reservation. The balance of the prepaid account has reached zero or a negative value, therefore the call cannot be authorized anymore. A non authorization event will be generated.

Pre Condition

- The end user has a prepaid account
- The end user places a voice call
- There is no sufficient balance for the duration of the reservation requested (according to policy of the SP).
