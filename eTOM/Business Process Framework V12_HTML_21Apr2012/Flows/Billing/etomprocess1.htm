(3) eTOM Process Type Mediate Resource Usage Records

Brief Description

Validate, normalize, convert and correlate usage records collected from the network.

Extended Description

The purpose of the Mediate Usage Records processes is to validate, normalize, convert and correlate usage records collected from various pieces of equipment in the network.
