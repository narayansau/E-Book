Billing Flow - Level 3 Scenario 2: Debit request

Diagram Description

The end user is finishing a call using a prepaid account. A debit request events was generated for charging the call, i.e; decreasing the balance. (see Figure 3). Balance has been previously reserved, the call is rated correctly and the balance is decreased, the reservation is released. There is enough balance for the duration of the call.

Pre Condition

- The end user uses a prepaid account
- The end user has ended a voice call
- There was sufficient balance for the duration of the call.
