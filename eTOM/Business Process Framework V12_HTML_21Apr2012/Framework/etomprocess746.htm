(4) eTOM Process Type Correlate Service Alarm Event Notifications

Process Identifier: 1.1.2.3.7.3

Category: (4) eTOM Process Type

Brief Description

This process encompasses the correlation of redundant, transient or implied service alarm event notifications with a specific “root cause” service alarm event notification and associated service alarm event record. This process correlates service alarm event notifications to planned outage notifications to remove false service alarm event notifications arising as a result of the planned outage activity.
