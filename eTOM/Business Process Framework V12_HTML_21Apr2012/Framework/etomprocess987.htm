(4) eTOM Process Type Investigate Billing Event Related Problem

Process Identifier: 1.1.1.14.4.2

Category: (4) eTOM Process Type

Brief Description

Investigate problems related to billing event records.

Extended Description

The purpose of the Investigate Billing Event Related Problem process is to investigate problems related to billing event records. Where reporting indicates problems with billing event records such as data format errors, configuration problems, system or network problems this process initiates and manages an investigation and follow-up of the indicated problems.
