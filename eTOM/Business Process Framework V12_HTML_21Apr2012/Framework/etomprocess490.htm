(4) eTOM Process Type Allocate Resource

Process Identifier: 1.1.3.2.1.4

Category: (4) eTOM Process Type

Brief Description

This process allocates specific resources in response to issued resource orders.Where the Allocate & Install Resource processes are requested by a resource order issued in response to a confirmed customer order, this process is responsible for allocating the specific resources required to satisfy the initiating resource order. Any previously reserved specific resources are marked as allocated.
