(4) eTOM Process Type Manage Service Performance Quality Data

Process Identifier: 1.1.2.4.1.1

Category: (4) eTOM Process Type

Brief Description

This process monitors and logs the received specific service performance quality data, compares the received specific service performance quality data to performance quality standards set for each specific service (available from the Service Inventory), detect performance quality threshold violations which represent specific service failures due to abnormal performance, and detects performance degradation for specific services which provide early warning of potential issues.
