(4) eTOM Process Type Correlate Resource Alarm Event Notifications

Process Identifier: 1.1.3.3.1.3

Category: (4) eTOM Process Type

Brief Description

Resource alarm event record correlation and filtering encompasses the correlation of redundant, transient or implied resource alarm event notifications with a specific “root cause” resource alarm event notification and associated resource alarm event record. This process correlates resource alarm event notifications to planned outage notifications to remove false resource alarm event notifications arising as a result of the planned outage activity.
