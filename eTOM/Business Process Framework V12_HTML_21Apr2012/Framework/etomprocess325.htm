(3) eTOM Process Type Manage Resource Exit

Process Identifier: 1.2.3.3.7

Category: (3) eTOM Process Type

Brief Description

Identify existing resource classes which are unviable and manage the processes to exit the Resource from the market.

Extended Description

The Manage Resource Exit processes identify existing resource classes which are unviable and manage the process to exit the Resource from the services they support. The processes analyze existing resource classes to identify economically or strategically unviable classes, identify products, services classes & customers impacted by any exit, develop specific exit or migration strategies, develop resource infrastructure transition and/or replacement strategies, and manage the operational aspects of the exit process. A business proposal identifying the competitive threats, risks and costs may be required as a part of developing the exit strategy. These processes include any cross-enterprise co-ordination and management functions to ensure that the needs of all stakeholders are identified and managed.
