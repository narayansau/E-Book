(3) eTOM Process Type Close Resource Order

Process Identifier: 1.1.3.2.7

Category: (3) eTOM Process Type

Brief Description

This process monitors the status of the order and changes the status to closed when it is completed.

Extended Description

The objective of the Close Resource Order processes is to close a resource order when the resource provisioning activities have been completed.
These processes monitor the status of all open resource orders, and recognize that a resource order is ready to be closed when the status is changed to completed.
