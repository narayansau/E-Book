(4) eTOM Process Type Generate Resource Performance Degradation Problem

Process Identifier: 1.1.3.4.5.1

Category: (4) eTOM Process Type

Brief Description

This process creates a new Resource Performance Degradation Report as a result of specific resource performance notifications undertaken by the Monitor Resource Performance processes, or at the request of analysis undertaken by other RM&O, SM&O or RM&O processes which detect that some form of deterioration or failure has occurred requires an assessment of the specific resource performance.
