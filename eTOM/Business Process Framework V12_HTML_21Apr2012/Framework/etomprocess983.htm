(4) eTOM Process Type Distribute Billing Event

Process Identifier: 1.1.1.14.2.2

Category: (4) eTOM Process Type

Brief Description

Distribute billing event records to other processes.

Extended Description

The purpose of Distribute Billing Event process is to distribute billing events to other processes which need access to billing events.
As a typical example, billing events are transferred to Charging process for event/product charging via this process. In general, the billing events are distributed in the specific format, e.g. plain text format, binary format, XML format. This process is also responsible for recording distribution logs to avoid duplicated billing event distribution.
